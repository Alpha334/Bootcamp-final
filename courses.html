Our Five Core Courses

Five intensive, parallel tracks running throughout the entire 8-week bootcamp. Every student takes all five courses simultaneously for a comprehensive, immersive experience.

📅 8 Weeks Total 📚 5 Parallel Courses ⏱ ~20 hrs/week 🎓 All Tracks Included
01

Web Development Fundamentals

⏱ Full 8-Week Program

Build a rock-solid foundation in the core technologies of the web. Learn to structure content with semantic HTML, style it beautifully with CSS, and add interactivity with vanilla JavaScript — practiced and reinforced throughout the bootcamp.

  • HTML5 semantic elements, forms, and accessibility
  • CSS3 selectors, box model, flexbox, and CSS Grid
  • Responsive design principles and mobile-first approach
  • JavaScript fundamentals: variables, functions, loops, conditionals
  • DOM manipulation and event handling
  • Debugging with browser DevTools
  • Weekly projects reinforcing each concept
  • Final: Build a responsive personal portfolio website

Technologies & Skills

HTML5 CSS3 JavaScript ES6+ Flexbox CSS Grid DevTools
02

Frontend Frameworks & React

⏱ Full 8-Week Program

Level up from vanilla JS to modern component-based architecture. Master React's declarative UI model, state management with hooks, and utility-first styling with Tailwind CSS — integrated into every week's work.

  • React components, JSX, and the virtual DOM
  • State management with useState and useReducer
  • Side effects with useEffect and custom hooks
  • Props, component composition, and lifting state up
  • React Router for single-page navigation
  • Tailwind CSS utility classes and customization
  • Weekly projects building React components
  • Final: Build an interactive task management dashboard

Technologies & Skills

React 18 JSX Hooks Tailwind CSS React Router Vite
03

Backend & APIs

⏱ Full 8-Week Program

Move to the server side from day one. Build RESTful APIs with Node.js and Express, handle authentication securely, and work with both SQL and NoSQL databases — skills you build on week after week.

  • Node.js runtime, modules, and npm ecosystem
  • Express.js routing, middleware, and error handling
  • RESTful API design principles and best practices
  • JWT authentication and password hashing with bcrypt
  • MongoDB with Mongoose ODM
  • PostgreSQL basics and relational data modeling
  • Weekly API-building challenges
  • Final: Build a full-featured REST API for a blog platform

Technologies & Skills

Node.js Express MongoDB Mongoose PostgreSQL JWT bcrypt
04

Full-Stack Integration

⏱ Full 8-Week Program

Connect frontend and backend into a cohesive application from the start. Learn deployment strategies, environment management, and CI/CD basics — applied to projects throughout the program.

  • Connecting React frontend to Express API (CORS, fetch/axios)
  • Environment variables and configuration management
  • Deploying frontend to Vercel/Netlify
  • Deploying backend to Render/Railway
  • Git branching strategies and pull request workflows
  • CI/CD with GitHub Actions basics
  • Weekly deployment milestones
  • Final: Deploy your first full-stack application live

Technologies & Skills

REST APIs Axios CORS GitHub Actions Vercel Render Docker (intro)
05

Capstone & Career Prep

⏱ Full 8-Week Program

Work on your capstone project from week one, iterating and improving as you learn. Polish your portfolio, practice interviewing, and build a job search strategy alongside your technical growth.

  • Capstone project planning and iterative development
  • Advanced React patterns and performance optimization
  • Testing with Jest and React Testing Library
  • Portfolio website design and content strategy
  • Technical interview prep: algorithms and system design
  • Behavioral interview techniques and storytelling
  • Resume writing, LinkedIn optimization, and networking
  • Final project presentation and peer review

Technologies & Skills

Jest React Testing Library Portfolio System Design Interview Prep Career Strategy
